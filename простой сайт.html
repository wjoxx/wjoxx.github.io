Синтаксис HTML5

HTML5 - это последняя версия языка разметки гипертекста, используемого для создания и структурирования веб-страниц.

Основные правила синтаксиса HTML5:

  1. Документ начинается с декларации(<!DOCTYPE html>)
  2. Структура документа:

    3. < html lang="ru" >
    < head >
    < meta charset="UTF-8">
    < title> Заголовок страницы </title>
    </head>
    < body>
    <!-- Контент страницы -->
    </body>
    </html>

< html > - корневой элемент, который может содержать атрибут lang для указания языка.
< head > - содержит метаданные, настройки и ссылки на стили или скрипты.
< body > - содержит содержание страницы.

Теги могут быть парными (с открывающим и закрывающим тегом), например: < p > Абзац текста </p>

Или одинарными (самозакрывающимися) - например, < img >, < meta >, < br >.

Атрибуты - это дополнительные параметры у тегов, например:
  • Ссылка: < a href="https://example.com">Ссылка
  • Картинка: < img src="image.jpg" alt="Описание">

Семантические теги

Семантические теги - это теги, которые отражают смысл содержимого, делают структуру документа более
понятной для людей, поисковых систем и вспомогательных технологий.
HTML5 ввёл множество новых семантических элементов для улучшения структуры:
< header > - заголовок, шапка страницы или раздела.
< nav > - навигационное меню.
< main> - главный контент страницы.
< section> - тематический раздел.
< article> - самостоятельная публикация, статья.
< aside> - дополнительный контент, боковая панель.
< footer > - подвал, нижняя часть страницы.

Особенности HTML5:
  • Отсутствие необходимости закрывать некоторые теги — браузеры умеют автоматически закрывать, но лучше писать закрывающий тег для понятности.
  • Поддержка новых мультимедийных элементов: < audio >, < video >, < canvas >
  • Встроенная поддержка форм с новыми типами < input >: email, date, color, range и др.
  • Работа с API и скриптами: < script >, < link > для подключения внешнего кода и стилей.
